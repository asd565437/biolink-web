# BioLink Frontend

## 專案介紹

BioLink Frontend 為 BioLink 平台的前端應用程式，採用 React 開發，負責使用者介面、互動流程及前後端資料串接，提供流暢的使用者體驗。

## 功能

- 使用者登入 / 註冊
- 個人資料管理
- 好友互動
- 即時互動功能
- 響應式介面

---

## 技術

### Frontend

- React
- React Router
- Axios

### UI

- HTML5
- CSS3
- JavaScript

### API

- RESTful API 串接

---

## 我的負責內容

- React 前端介面開發
- 元件(Component)設計
- 頁面路由管理
- 前後端 API 串接
- 使用者互動流程設計
- Socket.IO 前端整合
